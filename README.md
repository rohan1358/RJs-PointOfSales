## description

- Backend (use my repository which name is BackEnd-Point-Of-Sales for Backend)

## Installation

run this command on the terminal

```bash
git clone https://github.com/rohan1358/RJs-PointOfSales.git
cd RJs-PointOfSales
npm install
npm start
```
